# PixelForge 0.1.0-alpha.1 受邀测试（一页任务说明）

> 适用范围：macOS 受邀 alpha 候选。请独立完成，不接受口头教学；这不是公开发布或跨平台验收。

## 准备与计时

- 准备 1–5 张你平时会处理的 AI 图片；不要使用仓库内测试图片。
- 解压候选包并启动 `PixelForge.app`。若 macOS 阻止未签名应用，请记录为失败点，不要让开发者代操作。
- 从应用窗口首次出现开始计时；在下表记录第一次成功导出的用时。

## 15 分钟最小旅程

1. 从空画布导入图片，确认内容立即可见；再导入一批，确认可以找到新增内容。
2. 对图片执行 Cleanup 或 Batch Cleanup；观察进行中与完成反馈，并判断处理前后差异。按需要尝试 Matte、Slice 或 Outline。
3. 在批次卡中查看、筛选或标记结果，保留你愿意继续使用的素材。
4. 保存为 `.pxproj`，关闭应用，重新打开保存文件；确认内容和选择结果仍在。
5. 导出至少一个 PNG；若有多张，再导出 spritesheet。确认界面显示完整路径，并从 Open Folder 找到产物。
6. 修改项目但不要保存，尝试 New 或 Quit；确认可以选择 Save / Discard / Cancel，选择 Cancel 后工作仍在。
