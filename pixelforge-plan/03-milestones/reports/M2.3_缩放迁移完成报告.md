# M2.3 缩放迁移完成报告

日期：2026-06-16

## 1. 完成结论

M2.3 已完成可自动验证的缩放体系迁移：主窗口 chrome 改由 `Window.content_scale_factor` 继承缩放，UI 组件级 `ui_scale` / `_scaled_int()` 约定已从 `pixel/ui` 删除；画布美术改为按 `F_canvas = max(1, round(F))` 做反向补偿，保证 100% zoom 下美术像素落到整数物理像素。新增静态守护脚本会阻止旧缩放约定复活。

本轮未改 core 算法、图模型、项目格式、`02-contracts/`，不升 `.pxproj` 版本。
