## 最短复验

1. [ ] 在 Output 和长 Prompt 中滚到末端后继续滚动 10 次，确认画布不缩放，滚动条不跳端。
2. [ ] 打开 API 设置，粘贴一个完整 `/v1/chat/completions` 测试地址，确认协议自动切到
   Chat Completions；填写该中转站实际模型名和测试 Key，执行连接测试后保存，关闭并重开
   设置，确认地址、模型、协议和“Key 已保存”仍在。若提示模型列表无法确认，确认保存后的
   精确配置仍可在生成卡中选择；不要在本项发送图片生成请求。
3. [ ] 检查生成卡无内部滚动、比例锁、Quality、Advanced、Seed 和费用；确认四档分辨率、
   `16:9 横向 / 9:16 纵向 / 1:1 正方形` 三种生成比例及 1–16 数量可选，5–16 会先确认；
   模型行显示刚才保存的远端模型名，而不是按地址猜测的固定值。
4. [ ] 切换开发者模式，确认最终提示词预览只在开启时出现，并且前缀只出现一次。
5. [ ] 粘贴长中文、英文、日文和无空格文本，确认全部在输入框内换行，保存内容不变。
6. [ ] 点击风格提示词的“新建 / 编辑 / 重命名”，确认右侧检查器自动打开，名称和 prefix
   输入只出现在检查器；复制为用户预设后编辑、保存、删除，再保存/重开项目确认快照可重放。
7. [ ] 导入至少 12 张参考图，在 2560×1440、100% 画布倍率下确认图片明显可读；选中画布
   上两张图片并整组拖进 Reference Set，确认按选择顺序追加、源图片不移动且一次 Undo 可撤销。
   把鼠标从缩略图移到右上角 ↻ / ×，确认不闪烁；分别点击替换与删除并验证各自 Undo。
8. [ ] 把 Reference 接到空白 Output，运行选区；确认图片按顺序立即出现，没有 Queued/Running
   或生成状态。
9. [ ] 点击像素清晰“设置”，确认完整参数在右侧；修改参数、Undo，并确认只能从卡片 Footer
   开始运行，运行中参数不可编辑。
10. [ ] 创建三张重叠卡片并依次点击下层卡片，确认被点击卡片始终在视觉与命中上最靠前；
    保存重开后基础 z-order 未被临时提升污染。
