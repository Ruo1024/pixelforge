### ⚠️ 开始前必读——本版本 7 个已知限制（避免误判）

1. ❌ **W / M / L 手动选区不会进入 Matte / Slice / Outline**：选完区再点 Matte，仍是对**整张图**抠图，选区被忽略。"先手动选背景再抠"这条路当前走不通（选区只更新状态栏 + 画蚂蚁线）。
2. ❌ **切换工具时鼠标光标不变化**（计划要求过，未实现）。
3. ⚠️ **蚂蚁线只画选区的外接矩形**，不画魔棒/套索的真实不规则轮廓——看到的是个方框，属正常。
4. ⚠️ **批次菜单的 Matte / Outline 用固定参数**（抠图 flood+容差12+无羽化；描边 外描边+黑色），不能调。
5. ⚠️ **批次处理进度只有"运行中"状态**，没有"7/20"百分比。
6. ⚠️ **一次导入多张图只会进"批次卡"**（不单独落成 sprite），因此这些图**无法用 Matte/Slice/Outline 预览对话框**精修（那些要求恰好选中 1 个 sprite）。要精修请单张导入。
7. ⚠️ 顶栏 **New / Open / Save** 按钮与 **File 菜单**功能重复（File 菜单多一个 Import Images）。

## 3. 像素清洗台 / 检查器（M1，右侧面板）

> 选中**单张 sprite**时右侧面板收集参数，改动后约 0.3s 自动出**预览**，点 **Apply Cleanup** 才真正执行。这是把"原始图"清洗成"干净像素图"的核心面板。

### 含义速查（每个选项代表什么）
| 选项 | 代表什么 / 怎么影响结果 | 默认 |
|---|---|---|
| **Selection（计数）** | 当前选中素材数；为 0 时 Apply 禁用 | 0 |
| **Auto-detect grid** | 自动推断原图"每个像素块多大、偏移多少"。取消则用下面手填的 Scale/Offset | 勾选 |
| **Scale** | 像素块尺寸 / 降采样倍率：原图多少像素合成 1 个逻辑像素 | 4.0 |
| **Offset X / Y** | 网格起始偏移，用于对齐像素栅格 | 0 |
| **Run resample** | 是否执行"降采样到像素网格"这一步 | 勾选 |
| **Resample 模式** | 每个像素块如何取色：**Mode**众数(最稳，默认) / **Center**取中心点 / **Median**中位 / **Edge Aware**边缘感知 | Mode |
| **Run quantize** | 是否执行"减色/量化"这一步 | 勾选 |
| **Quantize 模式** | **Auto K**自动定色数 / **Fixed Palette**套固定调色板 / **None**不减色 | Auto K |
| **Auto-K strategy**（仅 Auto K 显示） | 选色算法：**Median Cut**中位切分(快) / **K-means**(慢，可能更准) | Median Cut |
| **Palette**（Fixed Palette 用） | 选内置调色板(如 db32)或自定义；最后一项"导入…"可载入 JSON 调色板；下方有色条预览；自定义可删 | db32 |
| **Max colors / K** | Auto K 模式下的最大颜色数上限 | 16 |
| **Dither（抖动）** | 用少量颜色模拟更多色阶：None / Bayer 2/4/8(有序点阵) / Chromatic(色度) / Error Diffusion(误差扩散) | None |
| **Strength** | 抖动强度 | 0 |
| **Chroma** | 色度抖动量（配合 Chromatic） | 0 |
| **Density** | 抖动覆盖密度 | 1 |
| **Apply Cleanup / Cancel Cleanup** | 执行 / 取消；运行中 Apply 禁用、Cancel 可用 | — |
| **Report** | 执行后显示：检测 scale、置信度、颜色数、输出尺寸、非方形告警 | — |

## 4. 抠图 Matte（M2，单张）

> 前提：**恰好选中 1 个 sprite**，点顶栏 **Matte** 打开对话框。

### 含义速查
| 选项 | 代表什么 | 默认 |
|---|---|---|
| **Mode 模式** | **Flood 泛洪**：从图像边缘只清除"和背景相连的外部"同色像素，**保留**物体内部同色高光/镂空。**Global 全图**：全图凡接近背景色都清，含物体内部同色区 | Flood |
| **Tolerance 容差（0–100）** | 颜色相似度阈值（OKLab 距离）。越大清得越狠（可能误伤边缘），越小越保守（可能残留背景） | 15 |
| **Feather 羽化（0–8）** | 边缘半透明过渡宽度（像素）。0=硬边 | 1 |
| **预览（220px）** | 拖动参数后约 0.18s 刷新的实时抠图预览 | — |
| **橙色告警** | 判定"非纯色底"（背景不是一块纯色）时显示，并**不修改图像** | — |
| **Apply / Cancel** | Apply 生成新素材放在原图右侧（不覆盖原图）；Esc/Cancel 无副作用 | — |
