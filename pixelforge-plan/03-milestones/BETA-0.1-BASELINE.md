## 1. 为什么重新归并

旧执行书按 M0–M7 逐功能扩张，最终形成过名为 `1.0.0-rc.1` 的工程候选。但项目所有者首次在 macOS/Godot 编辑器中整体试用后，发现首启弹窗重叠、字体截断/异常、跨显示器 UI 缩放不可信、无中文界面，以及无限画布仍偏向“流程摘要卡”，没有达到内容工作台预期。

因此，旧版本号只保留为 Git 与报告中的历史工程标签。产品路线从现在起重基线：M0–M7 的既有实现统一称为 **Beta 0.1 工程基线**；它不是人工通过的 Beta，也不是已发布的 v1.0。

## 4. 统一未关闭项

1. 截图已证明 `Set Up PixelForge` 存在内容重叠；`Recover Autosave`、工具栏、检查器和其他位置出现字体底部截断或异常。
2. 历史缩放方案没有完成 macOS 内屏、外屏、跨屏与不同缩放档的真实视觉矩阵；当前问题不得以旧“修复完成”标题关闭。
3. 现有 `ui.language = en` 未形成可用本地化链，也没有随包 CJK 字体和中文布局回归。
4. 批次卡可以容纳真实图片，但普通节点仍是固定尺寸摘要卡，关键参数依赖菜单/模态框；无限画布没有形成内容模块工作台。
5. M4-V1 真实 API 与三人 A/B、三平台完整旅程、强杀恢复 10/10、法务和陌生用户测试仍待验。
6. UX-4 overview 两轮未通过并已撤销，不能在新计划中倒填完成。
