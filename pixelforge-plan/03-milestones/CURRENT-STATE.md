- 日期：2026-07-12

## 触发 Beta 0.2 的项目所有者体验证据

项目所有者在 Beta 0.2 开工前的 macOS Godot Debug 运行中确认：

1. `Set Up PixelForge` 弹窗文字/控件重叠；
2. `Recover Autosave` 及其他位置存在文字底部截断或字形异常；
3. 不同显示器和缩放环境下的 UI 可靠性尚未被真实证明；
4. 当前无可用简体中文界面；
5. 无限画布底层可操作，但普通节点主要是摘要卡，只有批次卡真正承载内容，整体工作流没有达到类似 LibTV 的内容模块工作台预期。

Beta 0.2 已为上述问题提供自动化和 agent 工程证据，但项目所有者尚未重新签收，因此不能把它们改写为人工通过。旧 M0–M7 的状态仍为：`历史工程完成 / 部分人工证据 / 当前整体人工未通过 / 未发布`。

## 项目所有者最新范围决定

1. Beta 0.3–0.5 连续完成图像向无限画布原型，再统一人工验收；
2. 账号、权限、协作、共享和项目版本历史永久排除，不登记为欠债；
3. 本地模型与 ComfyUI 无限期延后，主界面隐藏但不在本轮删除底层；
4. 开发期间不使用 Computer Use 做完整测试，只运行自动化与脚本截图；
5. 真实云端、中文、跨屏和完整旅程在三版完成后由项目所有者统一测试；
6. 人工 `go` 前不合并 `main`、不 push。
