# M6 — 像素编辑器（功能5a：Aseprite-lite 修补工具）

> 目标：双击素材进入编辑模式：绘制/调色板/图层/动画时间线。定位"修 AI 产出的最后 30%"而非全功能 Aseprite（PRODUCT.md 边界）。
> 依赖：M0–M2（素材库、选区状态机、调色板）。M5 的动画模型与本里程碑时间线共用契约。
> 先例：Pixelorama 纯 GDScript 实现全功能像素编辑器——技术上限远高于本里程碑需求（RESEARCH-NOTES §1）。

---

## M6-1 编辑器外壳与文档模型

**目标**：编辑 Tab：打开素材 → 可编辑文档（图层栈+帧序列）→ 存回素材库。

**技术实现指导**：
- 文档模型 `PFEditDoc { size, layers: [PFEditLayer{name, frames: [Image], visible, opacity, locked}], frame_durations, palette }`——**帧×层矩阵**（Aseprite cel 模型简化版：暂不做 cel 链接/独立 cel 位移）。
- 入口：素材双击 → 单层单帧文档；动画素材 → 单层多帧；"作为图层导入"支持多素材叠层（修图常用：垫参考）。
- 存回：拍平可见层 → 覆盖原素材（经 undo 提示）或另存新素材（默认，provenance.origin="edited", parent 链接）。多帧 → 更新/创建 anim.json（M5-3 契约）。
- 编辑画布：独立场景复用 CameraRig + 像素网格 overlay（M0-3 组件抽取的回报点）；棋盘格透明底。

**验收标准**：
1. 打开→改→另存全链 provenance 正确；多层拍平顺序/不透明度正确（与 M5 渲染同一合成函数——抽公共 `compositor.gd`，单测覆盖）。
2. 文档脏标记/关闭确认/undo 独立栈（编辑器内 undo 与画布 undo 隔离）。

## M6-2 核心绘制工具组

**目标**：铅笔/橡皮/取色/油漆桶/直线/矩形/椭圆/选区移动，全部像素完美。

**技术实现指导**：
- 工具状态机复用 M2-2 框架（注册式：每工具一类，声明光标/快捷键/参数控件）。
- 铅笔：Bresenham 连线插值（快速拖动不断点）；**像素完美模式**（开关，默认开）：消 L 形冗余角（Aseprite pixel-perfect 算法：检测新画点与前两点构成 L 即删中点）。笔刷尺寸 1–8（方/圆）。
- 绘制实现：直接改 frame Image + 脏矩形 `update_region`（ImageTexture.update 局部，**不得**每笔重建纹理——架构 §7 预算 16ms）。
- 油漆桶：复用 M2 flood（容差 0 默认）；全局填充模式（同色全图替换）。
- 形状工具：预览层绘制（OverlayLayer），确认落笔合成进 Image（undo 快照=形状 bbox）。
- 取色 Alt 全工具通用；前景/背景色 + X 交换。
- 镜像绘制（水平/垂直对称）——像素画高频需求低成本实现（坐标镜像重放笔迹）。

**验收标准**：
1. 单测：Bresenham 与像素完美算法黄金用例（已知输入点序列→期望像素集）。
2. 1px 笔快速画圈无断点（模拟 120Hz 输入事件序列）；笔迹延迟人工实测 < 16ms（Profiler 截图存档）。
3. 全工具 undo/redo 正确（含选区移动落定）。

## M6-3 调色板面板与色板约束绘制

**目标**：编辑器与风格预设的色板联动（产品差异点：约束式修图防风格漂移）。

**技术实现指导**：
- 调色板面板：色块网格（项目预设色板预载）、最近使用色、添加/删除/重排（写素材级自定义板）。
- **约束模式**（默认开，可关）：拾色器仅限色板色；外部取色（参考层）自动吸附最近色板色（OKLab）+ 提示原色与吸附色对比。
- 色板编辑联动：改色板某色 → 可选"全图重映射该色"（索引语义的轻量实现：维护 color→pixels 反查缓存仅对 ≤64 色图启用）。
- "全图量化回色板"按钮（调 M1 quantizer，处理外部粘贴污染）。

**验收标准**：
1. 约束模式下任何工具无法产出板外色（属性测试：随机操作序列 1000 步后图颜色集 ⊆ 色板）。
2. 重映射 32×32 图 < 50ms（缓存路径）。

## M6-4 动画时间线

**目标**：帧管理 + 洋葱皮 + 实时预览，覆盖"给素材加 2–8 帧动效"场景。

**技术实现指导**：
- 时间线面板（底部停靠）：帧缩略图条 × 图层行矩阵；帧操作（添加/复制/删除/拖拽重排/时长 ms 编辑——双击数字）；当前帧高亮，左右方向键切换。
- 洋葱皮：前后各 N 帧（默认 1）半透明叠加（前红后蓝 tint 惯例）；性能：直接叠 TextureRect 层，帧切换才更新。
- 预览：播放按钮 + 速度倍率 + 循环开关；独立小窗预览（1x/2x/4x）可常开悬浮。
- 帧标签（tag）：命名区段（"walk" 1–4、"idle" 5–6）→ 导出 spritesheet 时 JSON 带 tag 元数据（M2-5 导出器扩展）。

**验收标准**：
1. 集成：8 帧行走循环编辑→洋葱皮辅助下逐帧调整→预览流畅→存回素材库→Board(M5) 中即时可用——跨里程碑契约（anim.json）回归测试。
2. 时间线 32 层×64 帧矩阵 UI 操作不卡（虚拟化滚动或上限提示，二选一明确实现）。

## M6-5 修图增强三件套（AI 工作流特化）

**目标**：针对"修 AI 图"的高频动作做快捷封装（产品差异化，Aseprite 没有的"AI 后处理语境"工具）。

**技术实现指导**：
- **杂色清扫**：选区/全图内，孤立色（出现次数 < n 或与邻域全不同的单像素）一键吸附邻域主色；预览高亮将变更像素。
- **断线修补辅助**：检测 1px 轮廓断点（形态学端点检测）高亮提示（不自动改——辅助人眼，像素画轮廓是命脉）。
- **局部重生成占位**：选区 → "发送到节点图"（生成 image_input(带 mask) 预连 ai_generate(inpaint) 的小图——M4 provider 支持 inpaint 时激活，否则灰显 tooltip 说明）。inpaint 全链路在 provider 能力可用后追加卡（backlog 标注，不阻塞本里程碑）。

**验收标准**：
1. 杂色清扫黄金用例（含"不该清的 1px 高光"反例——亮度显著性保护规则）。
2. 端点检测在断线 fixture 上召回 ≥ 90%、误报 ≤ 10%。
